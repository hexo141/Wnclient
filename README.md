# Wnclient
A cross-platform visual aid tool made with Python that masquerades as a vape
How to use?
1. Clone repository
   ```bash
   git clone https://github.com/hexo141/Wnclient.git
   cd Wnclient
   ```
2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
3. Run!!!
   ```bash
   python main.py
   ```
4. enjoy
